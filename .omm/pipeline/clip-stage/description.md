Optional zero-shot classifier. Loads open_clip ViT-B-32/laion2b_s34b_b79k, encodes text prompts from categories.yml plus a baseline anchor, encodes image batches, subtracts anchor, writes top-3 (name, score) pairs to clip_labels as JSON. Only runs when both torch and open_clip are importable and categories.yml exists. Batch halves on CUDA OOM. Per-image decode failures skip and log.
