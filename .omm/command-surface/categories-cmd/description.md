`picpic categories <library> --list|--check|--init` (required mutually-exclusive group). --list dumps parsed taxonomy. --check validates YAML syntax and reserved-name rules (rejects "未分类"). --init writes a default categories.yml if missing. Returns 2 on validation failure. Exits with the trailing defensive `return 2` if none of the three flags matched (unreachable via argparse, kept for safety).
