CLIP zero-shot classifier. Loads an open_clip model (default ViT-B-32/laion2b_s34b_b79k), encodes category prompts from categories.yml, encodes image batches, subtracts a baseline anchor embedding, and writes top-3 (name, score) pairs to photos.clip_labels as JSON. Batch size halves on OOM (RuntimeError containing 'out of memory'). Per-image decode errors are logged to stderr and skipped. Optional dependency probed via importlib.util.find_spec('torch') and find_spec('open_clip').
