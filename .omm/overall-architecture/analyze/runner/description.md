Coordinator for analyze passes. Accepts (conn, library, *, run_clip, force_clip, clip_only) — mutually-exclusive flags drive which passes execute. On CLIP failure catches CategoriesError/ClipUnavailable/FileNotFoundError/OSError/RuntimeError/AssertionError, prints to stderr, and returns Phase 1 results intact. Auto-writes default categories.yml on first analyze when [clip] extras installed.
