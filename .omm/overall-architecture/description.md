picpic is a local privacy-preserving photo triage tool. Users scan an on-disk photo library, run analyzers to extract features (hashes, EXIF, blur, near-duplicates, CLIP labels), apply rules that mark trash candidates, then review and dispose of photos through a local FastAPI web UI. All processing is local; originals are read-only for the lifecycle — only the trash module can move files, and only purge_trash can physically delete.
