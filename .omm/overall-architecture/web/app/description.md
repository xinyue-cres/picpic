FastAPI application. Endpoints: GET / (SPA index), GET /api/photos?tab=... (list active/candidates/trashed/labeled photos), GET /api/labels (category counts), GET /photo/{id} & /thumb/{id} (validated path serving), POST /api/trash/{id}, /restore/{id}, /purge. _photo_dict parses clip_labels JSON, filters malformed rows via isinstance dict check, and extracts top_label. NULL clip_labels counts as unclassified.
